# Find Pombon! — Deploy to Vercel

## Files in this folder
- src/App.js — the full game
- src/index.js — React entry point
- public/index.html — HTML shell
- package.json — project config

## Step 1 — Install Node.js
Go to https://nodejs.org — download and install the LTS version.

## Step 2 — Create a GitHub account
Go to https://github.com and sign up free.

## Step 3 — Upload to GitHub
1. Go to https://github.com/new
2. Name it: pombon-game
3. Set to Public → Create repository
4. Click "uploading an existing file"
5. Drag the entire pombon-game folder contents in
6. Click Commit changes

## Step 4 — Deploy on Vercel
1. Go to https://vercel.com → sign in with GitHub
2. Click "Add New Project"
3. Import your pombon-game repo
4. Click Deploy (Vercel auto-detects React)
5. Wait 60 seconds — done!

## Step 5 — Share on Twitter!
Your live link will be: https://pombon-game.vercel.app
